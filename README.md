# InterviewBridge Backend

## Project Overview
InterviewBridge is a production-ready interview management platform.

## Tech Stack
- Java 21
- Spring Boot
- Spring Security
- JWT
- PostgreSQL

## Features
- Authentication
- Authorization
- User Management
- Validation
- Global Exception Handling

## Setup

```bash
mvn clean install
mvn spring-boot:run
```
